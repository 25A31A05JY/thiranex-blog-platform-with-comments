# Blog Platform with Comments

## Description
A full-stack blogging platform where users can:
- Register and Login
- Create Blog Posts
- Edit/Delete Posts
- Comment on Blogs

## Technologies Used
- HTML
- CSS
- JavaScript
- Node.js
- Express.js
- MongoDB

## Features
- User Authentication
- Blog Management
- Comment System
- RESTful APIs

## How to Run Project

### Install Dependencies
npm install

### Start Server
npm start

### Open Frontend
Open index.html in browser
